선택지 제시 사용

LLM의 output을 출력하면서, 그 다음에 이어질 사용자의 입력을 예상하여 선택지를 제시하도록 합니다.

  • Optional parameter(선택적 파라미터)
  • default : 0 (선택지 제시 안함)
  • 1 이상의 값은 선택지의 개수를 의미합니다.

사용 예시

사용자 입력값
{
    "action": "sendmsg",
    "room_uid": "51VPKp3F0yPX",
    "user_id": "s8dfkjsdf",
    "msg": "안녕하세요",
    "api_key": "d4a3653a4ba72d6e04a293",
    "use_choices": 3
}
