AI 음성 기능 사용

Chldo.com에서 적용한 음성을 출력할 수 있습니다.

return_voice의 적용​

  • 입력을 하지 않는 경우, chldo 환경설정에서 기본 음성 반환 여부로 선택한 값을 기본으로 사용합니다.
  • return_voice
    • 0 : 음성 반환하지 않음
    • 1 : 음성 반환
